# Security Policy

## Supported Versions

The following versions of BOSC Community Library are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| < 1.1.x | :x:                |

## Reporting a Vulnerability

We take the security of our community resources seriously. If you find a security vulnerability, please do not report it in a public issue. Instead, please follow these steps:

1. Send an email to [security-placeholder@example.com] with the details of the vulnerability.
2. Provide a description of the issue and any steps to reproduce it.
3. We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.

Thank you for helping keep the BOSC community safe!
